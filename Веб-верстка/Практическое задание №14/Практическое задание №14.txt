1. Каким тегом добавляется в html элемент видео?
Ответ: Для добавления видео на страницу используется элемент <video>

2. Укажите основные атрибуты к этому тегу.
Ответ: Атрибуты элемента <video>:
• src - путь к файлу видео (абсолютный или относительный )
• controls - создание набора элементов управления воспроизведением (кнопки: воспроизведение, пауза, громкость)
• width/height - ширина/высота окна воспроизведения (в пикселях или %)
• autoplay - начать воспроизведение сразу же после завершения загрузки страницы
• loop - циклическое воспроизведение видеофайла
• poster - адрес изображения, которое браузер будет использовать, пока загружается видео или
пока пользователь не нажмет на кнопку воспроизведения
• muted - выключает звук при воспроизведении видеофайла
• preload - предварительная загрузка видеоконтента 

3. Как добавить на страницу сайта звуковой контент?
Ответ: Для добавления аудио на страницу используется элемент <audio>

4. Укажите основные атрибуты к звуковому тегу.
Ответ: Атрибуты элемента <audio>:
• src - путь к аудио файлу (абсолютный или относительный )
• controls - создание набора элементов управления воспроизведением (кнопки: воспроизведение, пауза, громкость)
• autoplay - начать воспроизведение сразу же после завершения загрузки страницы
• loop - циклическое воспроизведение аудио
•muted - выключает звук при воспроизведении аудиофайла
• preload - предварительная загрузка аудиоконтента
