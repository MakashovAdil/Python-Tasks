1. Какие функции выполняет модуль Random?
Ответ: Для создания массивов со случайными элементами служит модуль numpy.random


2. Как создать массив с помощью модуля?
Ответ: Самый простой способ задать массив со случайными элементами — использовать функцию sample.
Без аргументов возвращает просто число в промежутке [0, 1), с одним целым числом — одномерный массив, с кортежем — массив с размерами, указанными в кортеже 
С помощью функции randint или random_integers можно создать массив из целых чисел.


3. Назовите способ сделать случайную выборку из массива?
Ответ: Сделать случайную выборку из массива можно с помощью функции choice:numpy.random.choice(a, size=None, replace=True, p=None)


4. Что вообще такое - случайные числа?
Ответ: Случайные числа представляют собой произвольные данные, которые были получены в результате автоматической генерации компьютерной программой
